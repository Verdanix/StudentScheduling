# Student Scheduling
The Student Scheduling project aims to provide a simple solution to schedule shifts at any Student Run store.
The original project was privately developed and was complex to run.
This open-source version aims to simplify the codebase and make it easier for anyone to deploy and customize.

## Project Goals
### Primary Goal 1: Customizability
- **Exclusion of certain days**: Schedule generation should allow for the exclusion of specific days, such as holidays or special events.
- **Alternating shifts**: The system should support A/B day scheduling, allowing for alternating shifts based on predefined patterns.
- **Number of workers**: Users should be able to specify the number of workers required for each shift and the minimum number of shifts each person must work per month.
### Primary Goal 2: Output
- **Importable calendar**: Provide an output format that can be easily imported into all calendar applications, that support the iCal format.
- **Printable hall passes templates**: Generate printable hall passes for each scheduled shift, ensuring that students have the necessary documentation for their shifts.
